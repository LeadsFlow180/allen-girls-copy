# Artifact name supersession

**Effective:** 2026-07-14 (owner-approved)  
**Rule:** The word **Trinket** is retired for all platform-facing language.

## Current name

**The Astral Realm Stone**

Canonical source in code: `src/data/canon/astral-realm-stone.ts`

## Historical PDF

The file:

`AGA - Master Canon for Portal Origin, Nova Star Command, Elementia, SPARK, and The Astral Realm Stone.pdf`

is a **historical development record**. Its internal text may still say “Trinket” because that was the working name when the document was written.

For the live product (UI, scripts, voiceovers, placement, missions), always use **The Astral Realm Stone**.
